# homebrew-squad-cli

A Homebrew tap for installing the Squad CLI.

## Installation

1.  **Tap the repository:**

    ```bash
    brew tap peacockz/homebrew-squad-cli
    ```

2.  **Install the formula:**

    ```bash
    brew install squad-cli
    ```

## Usage

Once installed, you can use the `squad` command as you normally would. For example, to get help, you can run:

```bash
squad --help
```
